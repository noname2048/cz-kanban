# cz-kanban

@dnd-kit, redux, mobx 를 이용한 kanban board

![img.png](docs/img.png)

## 지원되는 기능

- 저장과 불러오기 기능 (localStorage)
- column 제목, task 제목 변경과 삭제
- column 간 순서 변경, task 간 순서 변경
- redux, mobx 를 이용한 상태관리 (각각)
